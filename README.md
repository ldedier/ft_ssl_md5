## supported hashes

* md5
* sha224
* sha256
* sha384
* sha512

## Usage

./ft_ssl  { md5 , sha256 , sha224 , sha384 , sha512 } `[-p]` `[-q]` `[-r]` `[-s string_to_hash]`

* ./ft_ssl hash -s string
* ./ft_ssl hash filename
